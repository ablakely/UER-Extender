UER Extender

UER Extender is an extension for Google Chrome that allows you to customize the sites interface.

Usage

Most features are self-explanatory. Due to current limitations, to use the db arrow keys you must use the ctrl button too.

Installing

Installing only requires 3 easy steps.

  1. Open Chrome and go to Setting > Extensions
  2. Tick the 'Developer Mode' box at the top left of the page.
  3. Drag and drop the uerextender.crx file to the extensions page.
